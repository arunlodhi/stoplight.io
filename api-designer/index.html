StopLight API Designer

The modern, collaborative API editor, built
around existing open standards.

No Lock In!

Design your API with our visual interfaces, and then easily export your definitions to Swagger or RAML to take advantage of the many other tools supporting these specs. We even generate Swagger + RAML on the fly, and make it available to you to access anytime, programatically, via our API.

Best in class JSON Schema Editor

JSON Schema is a powerful tool for defining and validating the structure of JSON data, but it can be cumbersome to write and maintain. Our JSON Schema editor was built from scratch to be the fastest, most intuitive way to design schemas.

Collaborate in Workspaces

Workspaces reduce development friction, and allow you to group together projects - like your internal and external APIs, microservices, etc - under one account. You can easily invite teammates to manage and collaborate on APIs.

Deep Prism Proxy Integration

The installable version of StopLight API Designer runs an instance of the Prism proxy on your computer, giving you local access to to all of it's mocking, logging, transformation, and validation features.

Save Time with API Discovery

Direct API traffic through the Prism instance running on your computer, and it can automatically discover and generate your API definitions for you. This includes full schemas for requests and responses!

HTTP Traffic Capture

Better debugging across your team with shared traffic logs and replay. Any HTTP traffic passed through the local Prism proxy can be captured and logged to the API designer.

Realtime

Never worry about referencing an out of date definition again. All edits and additions made in the dashboard are propogated, in realtime, to your teammates.
