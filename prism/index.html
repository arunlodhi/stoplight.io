Prism API Proxy

A lightweight, dependency free proxy, built to manipulate HTTP
traffic, and configurable via simple JSON files.

Sign Up

HTTP Traffic Manipulation

Quickly extend and transform existing APIs, without changing any of the underlying API code. Use our simple Javascript API to transform incoming requests, and outgoing responses. Add or remove response/request body properties, change headers, and more.

Request Routing

Granular control over which scripts, validation, etc, is run when. Configure Prism to run certain scripts for particular endpoints by describing the HTTP method + URL endpoint combinations in JSON.

HTTP Traffic Validation

Reduce errors, and make your APIs easier to consume with explicit error messages. Add your JSON schemas to your configuration files, and then use our Javascript API to have Prism validate traffic against those schemas, on the fly.

HTTP Logging

Better debugging across your team with shared traffic logs. Log traffic to disk using Prism's embedded file storage, or give it a Mongo URI and it will log traffic to your MongoDB. Use our Javscript API to only log certain requests, based on any conditions. You can also have Prism log to an HTTP location.

API Designer Integration

Use the StopLight API Designer to visually design and configure your API, and then have it generate your Prism configuration files for you.
