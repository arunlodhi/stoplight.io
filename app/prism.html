Prism API Proxy

A lightweight, dependency free proxy, built to manipulate HTTP
traffic, and configurable via simple JSON files.

Sign Up

HTTP Traffic Manipulation

Write middleware scripts, using our simple Javascript API, to transform incoming requests, and outgoing responses. Add or remove response/request body properties, change headers, and more.

Request Routing

Configure Prism to run certain scripts for particular endpoints by describing the HTTP method + URL endpoint combinations in JSON.

HTTP Traffic Validation

Add your JSON schemas to your configuration files, and then use our Javascript API to have Prism validate traffic against those schemas, on the fly.

HTTP Logging

Log traffic to disk using Prism’s embedded file storage, or give it a Mongo URI and it will log traffic to your MongoDB. Use our Javscript API to only log certain requests, based on any conditions. You can also have Prism log to an HTTP location.

Local API Access

Prism can run an API as well as its proxy, that exposes a few simple endpoints to retrieve any traffic that it has logged.

HTTP Mocking

Configure Prism to respond with example data for particular endpoints, mocking them out before you’ve built them into your real API.

Dependency Free

Prism is a single ~12mb file, that can run on most operating systems, and requires no dependencies. It’s extremely simple to setup.

Simple Configuration

Prism is configured via simple JSON files, that describe your endpoints, schemas, and manipulation scripts / functions.

API Designer Integration

Use the StopLight API Designer to visually design and configure your API, and then have it generate your Prism configuration files for you.

Utility Functions

Utility functions are custom functions that you can define once, and re-use in any of your endpoint middleware scripts.
