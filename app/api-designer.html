StopLight API Designer

The modern IDE for your APIs, built around existing open standards.

Sign Up

No Lock In!

Design your API with our visual interfaces, and then easily export your definitions to Swagger 2 or RAML. We even generate Swagger + RAML on the fly, and make it available to you to access anytime, programatically, via our API.

Best in class JSON Schema Editor

Our JSON schema editor was built from scratch to be the fastest, most intuitive way to edit schemas. The visual editor supports almost the entire JSON schema v4 spec. Of course, our editor generates valid JSON schema code that you can edit directly, whenever you want.

Collaborate in Workspaces

Workspaces allow you to group together projects - like your internal and external APIs, microservices, etc - under one account. You can easily invite teammates to manage and collaborate on APIs.

Prism Integration

The installable version of StopLight API Designer runs an instance of the Prism proxy on your computer, giving you local access to to all of itâ€™s mocking, logging, transformation, and validation features.

Save Time with API Discovery

Direct API traffic through the Prism instance running on your computer, and it can automatically discover and generate your API definitions for you. This includes full schemas for requests and responses!

HTTP Logging

Any HTTP traffic passed through the local Prism instance can be captured and logged to API designer, for debugging or later replay.

Realtime

All edits and additions made in the dashboard are propogated, in realtime, to your teammates. Never worry about referencing an out of date definition again.
